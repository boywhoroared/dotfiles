*shell.txt* Execute a shell command and load it's contents into a buffer 

====================================================================

USAGE                                                   *ShellUsage*

- Execute command using :Shell
- Re-Execute the command by typing <localleader>r in normal mode in a window opened by :Shell.
- Use <localleader>g go to the previous window.
- Use the command :Shell! reexecute the last command.

Adapted from http://vim.wikia.com/wiki/Display_output_of_shell_commands_in_new_window
